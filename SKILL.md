# Swarm Spawner - Skill Definition

**Purpose**: AI agent orchestration system that spawns short-lived ephemeral agents for single tasks, each optimized with model-per-subtask matching and Hedera blockchain auditability.

**Triggers**: 
- "spawn ephemeral agents", "swarm agent orchestration", "temporary AI agents"
- "parallel AI task execution", "stateless agent system"
- "Hedera agent audit", "PQC signed agents"

**Key Capabilities**:
1. **Ephemeral Agent Engine** - Spawn stateless agents that die after task completion
2. **Model Router** - Match subtask complexity to optimal model (GPT-4, Claude, Gemini, Ollama)
3. **Hedera Integration** - PQC signing, HCS audit trails, HTS credential minting
4. **Multi-language SDK** - TypeScript, Python, Rust, Go
5. **IDE Integrations** - Claude Code, Cursor, VS Code plugins

**Monetization**: Open Core (MIT) + Services (Professional $49/mo, Enterprise custom)
